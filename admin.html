Springer Certificate Course Data Admin

Admin tool - not intended for students
How this works: This page reads courses.xlsx from this Zope folder, parses the course data, and lets you download a fresh course-data.js file. You then upload that file back to this same Zope folder to update the student-facing tool. The file must be named exactly courses.xlsx (with a single sheet using the standard columns).
Step 01
Upload the Spreadsheet
In ZMI, upload your updated Excel file to this folder. The file must be named courses.xlsx and must have a single sheet with the standard column headers.
Step 02
Load & Preview
Click Load courses.xlsx below. This page will fetch the file from the server, parse it, and show you a preview of every course it found. Verify the data looks correct before proceeding.
Step 03
Download & Re-upload
Click Download course-data.js. Then go back to ZMI and upload that file to this same folder, replacing the old one. Students will see the new courses within 24 hours.
